# 💻 CodeLab Blog

Um blog básico e responsivo focado em tecnologia, desenvolvido para praticar os fundamentos de desenvolvimento web (HTML, CSS e JavaScript).

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes ferramentas:

- **HTML5**: Estrutura semântica do projeto.
- **CSS3**: Estilização personalizada, uso de **Flexbox** para layout e **Media Queries** para responsividade.
- **JavaScript**: Lógica de busca em tempo real e sistema de "favoritar" posts.
- **Google Fonts**: Integração da fonte **Space Grotesk** para um visual moderno.
- **SVG**: Ícones vetoriais para melhor performance e qualidade visual.

## 🛠️ Funcionalidades

- **Layout Responsivo**: O site se adapta perfeitamente a telas de computadores, tablets e celulares.
- **Busca Dinâmica**: Filtre os posts do blog em tempo real enquanto digita no campo de pesquisa.
- **Sistema de Favoritos**: Interatividade para marcar e desmarcar posts favoritos com feedback visual no ícone de coração.
- **Dark Mode Nativo**: Interface pensada para conforto visual com uma paleta de cores moderna.

## 🎨 Design

O layout foi inspirado em interfaces modernas de blogs de tecnologia, utilizando:

- **Cores principais**: `#17171A` (Header), `#121214` (Fundo) e `#E07B67` (Destaque/Coral).
- **Tipografia**: Space Grotesk.
